Project Information

  • Project: Neural Collaborative Filtering on Movielens Dataset
  • Date: Aug'23 - Oct'23
  • Project URL: github.com/martinnghoumin/ncf

Ensemble Learning to improve accuracy of predictions

Developed an NCF model to predict user ratings for movies. The model was trained on the Movielens dataset, and used an ensemble of deep learning and collaborative filtering techniques to improve accuracy.

  • constitutes Matrix Factorization, MLP & CNN models in ensemble, then fed to a NeuMF with ReLU activation
  • Achieved >74% accuracy on normal & time-weighted ratings
  • Removing CNN increases accuracy to >80%
